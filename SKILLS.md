# Project Skills & Capabilities

This application is equipped with several advanced features that make it stand out:

## 🍱 Menu Management
- **TikTok-style Scroll**: Vertical full-screen browsing experience.
- **Detailed Ingredients**: Every item includes comprehensive descriptions and ingredients in Arabic.
- **Shimmer Effects**: Professional loading states for a premium feel.

## 🤖 AI Intelligence
- **Persistent Memory**: Both Client and Admin assistants remember past interactions.
- **Context Awareness**: The AI is trained on the Azura menu and restaurant details.
- **Multi-API Support**: Automatic fallback between Groq and Pollinations.

## 📊 Analytics & Monitoring
- **Real-time Presence**: Admin can see active users and their current table.
- **Usage Depth**: Identifies "Heavy Users" based on cumulative seconds spent in the app.
- **Device Identity**: Uses `deviceId` to track users even without a formal account.

## 🔒 Security & Auth
- **Zero-Password Auth**: Simplifies the guest experience while maintaining identity.
- **Admin Access**: Protected via a master key (`azura2024`) with secure session management.
